# 🏆 SportSquad

Social media app for sports enthusiasts — squads, queuing, cost splitting, and more.

---

## 🚀 Deploy for Free (Step by Step)

### Option A — Vercel (Recommended)

1. **Create GitHub account** at github.com (if you don't have one)
2. **Create new repository** → name it `sportsquad` → click Create
3. **Upload these files** by dragging them into GitHub's file uploader
4. Go to **vercel.com** → Sign up with GitHub
5. Click **"Add New Project"** → Import your `sportsquad` repo
6. Leave all settings as default → click **Deploy**
7. ✅ Your app is live at `sportsquad.vercel.app`

---

### Option B — Netlify (Drag & Drop, no GitHub needed)

1. On your computer, open terminal in this folder and run:
   ```
   npm install
   npm run build
   ```
2. Go to **netlify.com** → Sign up free
3. Drag the generated **`build/`** folder onto the Netlify dashboard
4. ✅ Live instantly at a free `.netlify.app` URL

---

### Option C — Stackblitz (Fastest, works on mobile)

1. Go to **stackblitz.com/fork/react**
2. Delete the default `App.js` content
3. Paste the contents of `src/App.jsx`
4. Click **Share** → copy the public link
5. ✅ Done — shareable link, no account needed

---

## 💻 Run Locally

```bash
npm install
npm start
```
Opens at http://localhost:3000
